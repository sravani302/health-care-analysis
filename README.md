# health-care-analysis
Axon Healthcare Analysis | Excel, SQL, Power BI

Analyzed healthcare operational data to track patient visits, follow-up rates, and service utilization.

Extracted and cleaned data using SQL and Excel to ensure accuracy and consistency.

Built Power BI dashboards with KPIs such as total visits, follow-ups required, and completion rates.

Created DAX measures to calculate follow-up percentage and performance metrics.

Delivered actionable insights to support management decision-making and process improvement.

Prepared structured MIS reports for daily and monthly review.
